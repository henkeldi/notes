
# GDB

* Show source code
```(gdb) list```

* Set breakpoint at line
```(gdb) break 5```

* Run the program
```(gdb) run```

* Step

```(gdb) step```

* Show variable

```(gdb) p <variable-name>```

* Show memory location of variable

```(gdb) p &<variable-name>```

* Show memory

```(gdb) x/7tb 0x<address-in-hex>```
